# Algo-Fusion-DAA

A modern interactive platform for learning, visualizing, and understanding core **Design and Analysis of Algorithms (DAA)** concepts through real-time implementations, graphical demonstrations, and performance analysis.

---

## 🚀 Project Overview

Algo-Fusion-DAA is a web-based educational project designed to simplify complex algorithms using:

- Interactive visualizations
- Step-by-step execution flow
- Algorithm performance comparison
- Clean and responsive UI
- Real-world implementation logic

The platform helps students, beginners, and enthusiasts understand how algorithms work internally rather than only learning theory.

---

## ✨ Features

### 🔹 Sorting Algorithms

Visualize and compare sorting techniques with animated execution.

Implemented Algorithms:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

### 🔹 Graph Algorithms

Interactive graph traversal and shortest path analysis.

Implemented Algorithms:

- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra’s Algorithm
- Prim’s Algorithm
- Kruskal’s Algorithm

### 🔹 Dynamic Programming & Greedy Algorithms

Understand optimization techniques and problem-solving strategies.

Implemented Algorithms:

- Knapsack Problem
- Fibonacci Optimization
- Coin Change Problem
- Activity Selection
- Greedy Scheduling

### 🔹 Additional Features

- Responsive and modern interface
- Real-time algorithm execution
- Time complexity understanding
- Educational testing modules
- Modular code structure
